1. Einführung & Grundkonzept

Einführung & Grundkonzept

  • SVMs (Support Vector Machines) sind eines der bekanntesten Modelle im Machine Learning.
  • Sie sind besonders für klein- und mittelgroße Datensätze geeignet.
  • SVMs sind flexibel und können für Klassifikations- und Regressionsprobleme verwendet werden.
  • Im Klassifikationskontext sind SVMs diskriminative Klassifikatoren.

Unsere Ausgangssituation

  • Wir betrachten zwei Klassen in einem 2D-Feature-Raum.
  • Ein Klassifikator soll diese beiden Klassen voneinander trennen.
import matplotlib.pyplot as plt
from sklearn.datasets import make_blobs

X, y = make_blobs(n_samples=100, centers=2, random_state=2, cluster_std=1.2)
plt.scatter(X[:, 0], X[:, 1], c=y, s=50, cmap='autumn', edgecolors='gray')

Margin Maximierung

  • Anstatt nur eine einfache Trennlinie zu nutzen, wird um die Gerade ein Margin gelegt.
  • Dieser Margin reicht bis zu den nächstliegenden Punkten der Klassen.
  • Die optimale Trennlinie ist diejenige mit dem größten Margin.
  • SVMs sind deshalb Maximum Margin Estimators.
import numpy as np
xfit = np.linspace(-3, 2)

plt.scatter(X[:, 0], X[:, 1], c=y, s=50, cmap='autumn', edgecolors='gray')

for m, b, d in [(-1.5, -6, 0.33), (0.5, -5, 0.55), (-0.1, -5.5, 0.2)]:
    yfit = m * xfit + b
    plt.plot(xfit, yfit, '-k')
    plt.fill_between(xfit, yfit - d, yfit + d, edgecolor='none', color='#AAAAAA', alpha=0.4)

plt.xlim(-3, 2)

Entscheidungsfunktion

  • Für die Visualisierung der SVM nutzen wir die decision function.
  • Sie zeigt Entscheidungsgrenze und Margin-Linien im 2D-Raum.
  • Die Support Vectors werden dabei besonders hervorgehoben.

    def plot_svc_decision_function(model, ax=None, plot_support=True):
    """Plot the decision function for a 2D SVC"""
    if ax is None:
        ax = plt.gca()
    xlim = ax.get_xlim()
    ylim = ax.get_ylim()

    x = np.linspace(xlim[0], xlim[1], 30)
    y = np.linspace(ylim[0], ylim[1], 30)
    Y, X = np.meshgrid(y, x)
    xy = np.vstack([X.ravel(), Y.ravel()]).T
    P = model.decision_function(xy).reshape(X.shape)

    ax.contour(X, Y, P, colors='k', levels=[-1, 0, 1],
               alpha=0.5, linestyles=['--', '-', '--'])

    if plot_support:
        ax.scatter(model.support_vectors_[:, 0], model.support_vectors_[:, 1],
                   s=300, linewidth=1, facecolors='none')

    ax.set_xlim(xlim)
    ax.set_ylim(ylim)

SVC in sklearn

  • SVC bedeutet Support Vector Classifier.
  • Eine SVM kann nicht nur klassifizieren, sondern auch regressiv eingesetzt werden.
  • Beim Erstellen einer SVC wird der Kernel über kernel=<kernel_type> festgelegt.
  • Der Regularisierungsparameter C steuert den Kompromiss zwischen Margin-Breite und Fehlklassifikationen.
  • .fit(X, y) trainiert das Modell und findet die Trennlinie mit maximalem Margin.
from sklearn.svm import SVC

fig, axi = plt.subplots(1, 1)
model = SVC(kernel='linear').fit(X, y)
axi.scatter(X[:, 0], X[:, 1], c=y, s=50, cmap='autumn', edgecolors='gray')
plot_svc_decision_function(model, axi)
axi.scatter(model.support_vectors_[:, 0], model.support_vectors_[:, 1],
            s=300, lw=1, facecolors='none', edgecolors='blue')
axi.set_title('C = {:.1f}'.format(1.0), size=14)

Support Vectors

  • Der Fit des Modells hängt nur von den Support Vectors ab.
  • Diese Punkte liegen am Rand des Margins und bestimmen die Lage der Trennlinie.
  • Das Attribut model.support_vectors_ enthält diese Punkte direkt.
