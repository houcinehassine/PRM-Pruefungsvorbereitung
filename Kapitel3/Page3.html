Hard Margin vs. Soft Margin

Bei SVMs kann man den Margin entweder hart (hard) oder weich (soft) definieren.

Die Leitfrage lautet: Dürfen Datenpunkte innerhalb des Margins liegen – und wenn ja, in welchem Ausmaß?

Der zugehörige Hyperparameter ist in de>sklearn das Argument C. Der optimale Wert hängt vom Datensatz ab und wird typischerweise per Cross-Validation bestimmt [web:32][web:38].

Interpretation des Parameters C

>Große Werte von C entsprechen einem strengen, fast hard-margin-artigen Modell. >Kleine Werte von C erlauben mehr Verletzungen des Margins und machen das Modell weicher. >C ist also ein Regularisierungsparameter, der den Trade-off zwischen Margin-Größe und Fehlertoleranz steuert [web:32][web:38].

Slack Variable

In der Mathematik wird die Toleranz über die Slack-Variable \(\xi_i\) formuliert.

\[ \min_{\mathbf{w}, b, \xi} \frac{1}{2}\|\mathbf{w}\|^2 + C \sum_i \xi_i \]

mit den Bedingungen:

\[ y_i(\langle \mathbf{w}, \mathbf{x}_i \rangle + b) \geq 1 - \xi_i \] \[ \xi_i \geq 0 \]

Damit werden Margin-Verletzungen zugelassen, aber mit einer Strafe belegt [web:24][web:26][web:30].

Regularisierung und Trade-off

Der Parameter C stellt einen Trade-off zwischen „großem Margin“ und „Verletzung des Margins“ her.

Genau deshalb nennt man diesen Term auch Regularisierung: Margin-Maximierung ist eine Form von Regularisierung [web:30][web:35].

Warum nicht Accuracy?

Eine naheliegende Loss Function wäre die Accuracy, also die Übereinstimmung von Vorhersagen und Labels.

Diese ist hier jedoch ungeeignet, weil es sich um ein kombinatorisches Optimierungsproblem handelt. Deshalb braucht man eine kontinuierliche Loss Function.

Hinge Loss

Eine passende Verlustfunktion ist die Hinge Loss.

\[ L(t)=\max\{0,1-t\} \]
\[ t=y f(\mathbf{x})=y(\langle \mathbf{w}, \mathbf{x} \rangle + b) \]

Sie bestraft Punkte, die innerhalb des Margins liegen oder falsch klassifiziert sind [web:26][web:28][web:35].

Endgültiges Optimierungsproblem

Damit ergibt sich das letztlich optimierbare Problem zu:

\[ \min_{\mathbf{w}, b} \frac{1}{2}\|\mathbf{w}\|^2 + C \sum_i \max\{0, 1 - y_i(\langle \mathbf{w}, \mathbf{x}_i \rangle + b)\} \]

Das ist die Soft-Margin-Formulierung der SVM [web:24][web:30][web:35].
