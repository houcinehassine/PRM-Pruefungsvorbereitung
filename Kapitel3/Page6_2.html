6.2 Fault Classification – Kugellager

Im Kugellager-Beispiel werden Zeitreihensignale aus mehreren Dateien geladen, gestackt und anschließend in Fenster aufgeteilt. Ziel ist es, den Zustand des Lagers datengetrieben zu klassifizieren [web:76][web:79].

Die Daten werden aus einem Google-Drive-Verzeichnis geladen. Danach werden die einzelnen Signale in ein gemeinsames Zeitreihen-Array zusammengeführt.

Schritt 2: Signale in Fenster aufteilen

Die Zeitreihen werden in 100 Teilstücke zerlegt, damit später pro Fenster Features berechnet werden können. 

split_array = np.array_split(timeseries_array, 100)

Schritt 3: Features berechnen

Aus jedem Fenster werden typische statistische Merkmale berechnet: Standardabweichung, Skewness und Kurtosis. 

features_array = np.zeros((100, 4, 3))
for nsplit, split in enumerate(split_array):
    features_array[nsplit, :, 0] = split.std(axis=0)
    features_array[nsplit, :, 1] = skew(split)
    features_array[nsplit, :, 2] = kurtosis(split)

Schritt 4: Reshaping – Erklärung

reshape ändert nur die Form eines Arrays, nicht die enthaltenen Werte. Entscheidend ist, dass die Gesamtzahl der Elemente gleich bleibt. 

a = np.arange(60).reshape(5, 4, 3)
print(a)  # 5x4x3 = 60 Elemente

a.reshape(20, 3)  # 20x3 = 60 Elemente
