Hyperebene & Bias

Symbol / Formel Bedeutung
\(\langle \mathbf{w}, \mathbf{x}_i \rangle + b = 0\) Datenpunkt auf der Hyperebene [file:23]
\(\mathbf{w}\) Normalenvektor der Hyperebene [file:23]
\(b\) Bias (Abstand vom Ursprung) [file:23]

Trennungsbedingung

Formel Beschreibung
\(y_i = \operatorname{sgn}(\langle \mathbf{w}, \mathbf{x}_i \rangle + b)\) Modell-Vorhersage [file:23]
\(y_i(\langle \mathbf{w}, \mathbf{x}_i \rangle + b) \geq 0\) Korrekte Trennung [file:23]
\(y_i(\langle \mathbf{w}, \mathbf{x}_i \rangle + b) \geq 1\) Margin-Bedingung (skaliert) [file:23]

Margin & Optimierung

Formel Beschreibung
\(r = \frac{1}{\|\mathbf{w}\|}\) Margin-Radius [file:23]
\(\min \frac{1}{2}\|\mathbf{w}\|^2\) Minimierungsproblem [file:23]
\(\langle \mathbf{w}, \mathbf{x}_a \rangle + b = 1\) Skalierung auf Support Vector [file:23]

Slack Variable & Hinge Loss

Formel Beschreibung
\(\min \frac{1}{2}\|\mathbf{w}\|^2 + C \sum_i \xi_i\) Slack-Variable-Optimierung [file:23]
\(y_i(\langle \mathbf{w}, \mathbf{x}_i \rangle + b) \geq 1 - \xi_i\) Slack-Bedingung mit \(\xi_i \geq 0\) [file:23]
\(L(t)=\max\{0,1-t\},\; t=y(\langle \mathbf{w}, \mathbf{x} \rangle + b)\) Hinge Loss [file:23]
\(\min \frac{1}{2}\|\mathbf{w}\|^2 + C \sum_i \max\{0,1-y_i(\langle \mathbf{w}, \mathbf{x}_i \rangle + b)\}\) Finale SVC-Kostenfunktion [file:23][file:12]

SVR (Support Vector Regression)

Formel Beschreibung
\(\min \frac{1}{2}\|\mathbf{w}\|^2 + C \sum_i \max\{0,\|\;y_i-(\langle \mathbf{w}, \mathbf{x}_i \rangle + b)\;\|-\varepsilon\}\) SVR-Kostenfunktion [file:23]
SVR(kernel='rbf', C=..., epsilon=...) sklearn-Instanziierung [file:23][file:19]

Hyperparameter-Übersicht

Parameter Wofür? Groß = ? Klein = ?
C Regularisierung, Trade-off zwischen Margin und Fehler [file:23] Hard Margin, wenig Fehler erlaubt [file:23] Soft Margin, mehr Fehler erlaubt [file:23]
kernel Wahl des Kernels [file:23] 'linear', 'rbf', 'poly' [file:23][file:19] –
gamma (RBF) Wie stark ein Punkt auf andere wirkt [file:23] Enger Radius, eher lokal Weiter Radius, eher global
epsilon (\(\varepsilon\)) (SVR) Breite des Margin-Rohrs [file:23][file:19] Breites Rohr, weniger Support Vectors [file:19] Enges Rohr, mehr Support Vectors [file:19]
degree (Polynomial) Grad des Polynoms [file:19] Komplexere Entscheidungsgrenze [file:19] Einfachere Grenze [file:19]
lr (\(\eta\)) Lernrate bei Gradientenabstieg [file:23][file:12] Schnelle, aber instabile Updates [file:12] Langsame, stabile Updates [file:12]
epochs Anzahl der Trainingsiterationen [file:12] Längeres Training [file:12] Schnelleres Training [file:12]

Code-Snippets Schnellzugriff

SVC instanziieren und trainieren

from sklearn.svm import SVC
model = SVC(kernel='rbf', C=1.0)  # oder 'linear', 'poly'
model.fit(X_train, y_train)
predicted = model.predict(X_test)

SVR instanziieren und trainieren

from sklearn.svm import SVR, LinearSVR
model = SVR(kernel='rbf', C=100, epsilon=0.1, gamma='scale')
model.fit(X_train, y_train)
predicted = model.predict(X_test)

GridSearchCV für Hyperparameter-Tuning

from sklearn.model_selection import GridSearchCV
param_grid = {'C': [0.1, 1, 10], 'gamma': [0.001, 0.01, 0.1]}
clf = GridSearchCV(SVC(kernel='rbf'), param_grid, n_jobs=-1)
clf.fit(X_train, y_train)
