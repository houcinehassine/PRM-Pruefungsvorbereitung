6.3 Handschrifterkennung – Digits Dataset

Das Digits-Dataset enthält 8×8-Bilder handgeschriebener Ziffern. Es ist ein klassisches Multi-Class-Beispiel für SVMs [web:78][web:81].

Schritt 1: Digits-Dataset laden

Mit load_digits(return_X_y=True) erhält man direkt die Datenmatrix und die Zielwerte. Die Form ist typischerweise (1797, 64) für die Features und (1797,) für die Labels [web:78][web:81]. 

from sklearn.datasets import load_digits
from sklearn.svm import SVC
from sklearn.model_selection import train_test_split
import matplotlib.pyplot as plt

X, y = load_digits(return_X_y=True)
print(X.shape)  # (1797, 64)
print(y.shape)  # (1797,)

plt.imshow(X[15, :].reshape(8, 8), cmap=plt.cm.gray_r)
plt.figure()
plt.imshow(X[200, :].reshape(8, 8), cmap=plt.cm.gray_r)

Schritt 2: Train/Test Split und SVC

Nach dem Split wird ein Standard-SVC trainiert und auf den Testdaten ausgewertet. Danach kann eine einzelne Vorhersage direkt visualisiert werden [web:78][web:38]. 

X_train, X_test, y_train, y_test = train_test_split(X, y)
print(X_train.shape)  # (1347, 64)
print(X_test.shape)   # (450, 64)

model = SVC()
model.fit(X_train, y_train)

predicted = model.predict(X_test)

ndigit = 440
plt.imshow(X_test[ndigit].reshape(8, 8), cmap=plt.cm.gray_r)
plt.title(f'Predicted: {predicted[ndigit]}')

Ein eigenes Bild wird geladen, auf 8×8 skaliert, als Graustufenbild normalisiert und anschließend vom Modell klassifiziert [web:78]. 

import numpy as np
import matplotlib.pyplot as plt
from PIL import Image
